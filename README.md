# 🏥 Hospital ER Dashboard – Tableau Data Visualization Project

![Dashboard Preview](https://raw.githubusercontent.com/Chisom16/hospital-er/01d795c2e655e407175fbf8091be0e1f76bc1484/Hospital%20ER.png)

## 📊 Project Overview
In this project, I built an interactive **Tableau dashboard analyzing hospital emergency room (ER) operations**. The goal was to explore patterns in ER visits, patient wait times, satisfaction scores, and demographic trends while designing a clear and user-friendly analytical interface.

Rather than creating isolated charts, I focused on building a **fully interactive dashboard** that allows users to explore operational performance and patient experience through dynamic filters and multiple visualization types.

This project demonstrates my ability to transform raw healthcare data into an intuitive and visually engaging dashboard for exploratory analysis.

## 🎯 Key Questions Explored

This dashboard was designed to answer several operational questions:

- When are emergency room visits at their highest?
- Which departments experience the longest wait times?
- Does longer waiting affect patient satisfaction?
- What demographic groups visit the ER most frequently?
- How does patient satisfaction change over time?

---

## 📈 Dashboard Features

The dashboard includes multiple interactive visualizations designed to support exploratory analysis:

### ER Visits Trend
A **line chart** showing how ER visits change over time, helping identify trends or peak periods.

### Average Wait Time by Department
A **bar chart** highlighting departments with higher average wait times.

### Wait Time vs Patient Satisfaction
A **scatter plot** analyzing the relationship between operational efficiency and patient experience.

### Patient Age Distribution
A **histogram** showing which age groups most frequently visit the ER.

### Demographic Breakdown
A **stacked bar chart** analyzing ER visits by race and gender.

### Satisfaction Trend Over Time
An **area chart** tracking how patient satisfaction changes over time.

## 🎛 Interactive Dashboard Features

To make the dashboard dynamic and user-driven, I implemented several interactive elements:

- Filters for:
  - Patient Gender
  - Patient Race
  - Department Referral
  - Age Range
  - Date Range

- Charts that function as **interactive filters**
- A **clean container-based layout** for better usability
- Consistent color schemes for improved readability

These features allow users to explore the dataset from multiple perspectives within a single dashboard.
